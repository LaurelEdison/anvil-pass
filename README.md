# Anvil Password Manager

A password manager written in Rust

The goal of this project is to make a keepass(kdbx) compatible password manager, usable through CLI, TUI, GUI, and eventually GUI via tauri

[!NOTE]
Anvil currently writes KeePass KDBX4 databases. Support for additional database versions may be added in the future.

## Status

- Core vault library
- Entry & group management
- Save/load support
- Tests for core functionality

## Planned

- CLI interface
- TUI interface
- GUI (Tauri)
- Search improvements
- More KeePass features
